# PythonCMS

A content managemnt system to suit all of your needs! If your needs involve creating, reading, updating, or deleting content then this software is for you. Developed using Python and its webframework, Django, this exemplary webapp can handle everything to make your online prescence a snap!

# Technologies
### Back End
- [Google Cloud SQL](https://duckduckgo.com/?t=canonical&q=google+cloud+sql&atb=v207-1&ia=web)
  - Fully managed, scalable MySQL service
- [Django](https://www.djangoproject.com/)
  - Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design.
 
 
 ### Front End
 - [Bootstrap](https://getbootstrap.com/)
  - Bootstrap helps quickly design and customize responsive mobile-first sites
